# TessTwoDemo
基于tess_two做的Android文字识别demo。

简书文章地址：http://www.jianshu.com/p/cc9ae05423a8

用 https://github.com/oginotihiro/cropview  做了自己截图识别的功能，还不能自由截图。

最近加了一个自己训练的识别库，用来测试的，只能识别没几个字。可以跟原来的中文识别库比较，识别速度提高非常大，有需要的话可以自己训练识别库的。
